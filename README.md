# Учебный проект: Burger Fast (Верстка лендинга)

Этот проект представляет собой адаптивную верстку лендинга для вымышленного сервиса быстрого питания "Burger Fast". Сайт состоит из типовых для посадочной страницы блоков: шапка с навигацией, приветственная секция (hero), список блюд, секция с акционным предложением и карточки товаров.

Проект создан в учебных целях для отработки навыков верстки по макету из Figma, использования семантической верстки и базовых стилей CSS.

## Обзор проекта

### Функциональность
- **Шапка сайта (Header)**: Логотип, навигационное меню (пункты HOME, PAGES, OUR OFFER, PRICING, SHOP) и кнопка призыва к действию "Order now". Для активного пункта меню "HOME" применен стиль `active`.
- **Приветственная секция (Welcome)**: Крупный заголовок, подзаголовок, описание и кнопка для заказа. Секция занимает всю высоту экрана и имеет фоновое изображение.
- **Список блюд (Food List)**: Сетка из четырех карточек с изображением, названием и кратким описанием блюда.
- **Секция заказа (Order)**: Блок с крупным изображением бургера, заголовком, описанием, старой и новой ценой и кнопкой заказа.
- **Секция продуктов (Product)**: Сетка из четырех карточек товаров со скидкой. Каждая карточка имеет фоновое изображение, название, описание, цены и кнопку. Карточки выровнены таким образом, что текст в них расположен то слева, то справа для создания ритма.

## Процесс работы

### Технологии
- **HTML5**: Использована семантическая разметка (`header`, `main`, `section`), правильная структура документа.
- **CSS3**:
  - Flexbox (`display: flex`) для построения сеток и выравнивания элементов.
  - Подключение внешних шрифтов (Google Fonts: Roboto).
  - Каскадность и переопределение стилей.
  - Использование CSS-переменных (не реализовано, но может быть добавлено).
  - БЭМ-методология для именования классов (например, `header__logo`, `food-list__card`).

### Планы по доработке
- [ ] **Адаптивность**: Добавить медиа-запросы для корректного отображения на планшетах и мобильных устройствах.
- [ ] **Интерактивность**: Реализовать плавную прокрутку к разделам по клику на пункты меню, добавить состояния `:focus` для доступности.
- [ ] **Добавить футер**: В макете, вероятно, есть подвал сайта, который не был сверстан.
- [ ] **Оптимизация**: Сжать изображения, добавить `alt` тексты для всех картинок.
- [ ] **Доработка деталей**: Уточнить отступы, цвета и размеры в соответствии с макетом Figma, так как текущая верстка может иметь небольшие расхождения.
